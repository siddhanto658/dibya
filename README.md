# Steve FX - Professional Video Editor Portfolio

A modern, interactive portfolio website for professional video editors featuring a stunning visual design, smooth animations, and an AI-powered chatbot.

![Portfolio Preview](https://images.unsplash.com/photo-1521133573892-e44906baee46?q=80&w=1200&auto=format&fit=crop)

## Features

- **Immersive Hero Section** - Massive animated title with scroll-driven reveal
- **3D Coverflow Gallery** - Interactive project showcase with smooth transitions
- **AI Chatbot** - Groq-powered assistant for instant visitor engagement
- **Premium Visual Effects** - Animated blobs, custom cursor, and timeline playhead
- **Fully Responsive** - Looks great on all devices

## Tech Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **AI**: Groq API (LLama 3.1)
- **Deployment**: Vercel

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run locally: `npx vercel dev`

## Environment Variables

Create a `.env` file:
```
GROQ_API_KEY=your_groq_api_key
```
